# React 前端

基于 Stitch 生成页 `学迹智配管理后台` 的视觉风格实现。

## 启动

```powershell
cd frontend-react
npm install
npm run dev
```

默认端口：`5178`

## 路由

- `/login`：登录页
- `/`：工作台
- `/materials`：学习资料
- `/knowledge`：个人知识库 RAG
- `/videos`：视频复习
- `/jd-analysis`：JD 分析
- `/resume`：简历适配
- `/agent-tasks`：Agent 任务入口，当前不实现编排
- `/settings`：系统设置
